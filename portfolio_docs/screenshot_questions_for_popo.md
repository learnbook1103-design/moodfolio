## 🚀 5. 특정 시나리오별 질문

### 5-1. "Mumu 도슨트 모드" 스크린샷
- **촬영 시점**: 
  - 첫 인사 메시지가 나온 상태?
  - 버튼 선택지가 보이는 상태?
  - 답변이 나오고 다음 질문 버튼이 나온 상태?
- **강조할 포인트**: 
  - 입력창이 없다는 점?
  - 자동 스크롤 기능?

### 5-2. "관리자 대시보드" 스크린샷
- **촬영 시점**:
  - 템플릿 리스트 전체 뷰?
  - 토글 버튼 클릭 직전/직후 (GIF)?
  - 상태 변경 후 성공 메시지 토스트?
- **보안 고려사항**:
  - API 키나 민감한 정보가 화면에 노출되나요?

### 5-3. "게스트 → 회원 전환" 플로우
- **촬영 시점**:
  - 게스트 모드에서 포트폴리오 작성 중
  - 회원가입 버튼 클릭
  - 로그인 후 데이터가 유지된 화면
- **어떻게 증명할까요?**:
  - LocalStorage 개발자 도구 화면도 함께?
  - "데이터가 저장되었습니다" 토스트 메시지?
